Setup a pool for our service and put the configuration in dst.

Parameters
dstWhere to store the configuration.
PoolSizeThe size of the Pool, i.e. how many garbled gates are to be in the pool.
BufferSizeThe size of the Buffer, i.e. how many garbled gates are to be in the buffer.
CheckRateTHe probability that a freshly garbled gate is checked.
BucketSizeThe size of the Bucket for this pool, i.e. how many garbled gates are to be in a bucket.
Gate_Pointer_Sizeis the space needed to config the underlying protocol.
Bucket_Pointer_Sizeis the space needed to store a bucket.
PoolPRGSeedis the random seed used in the protocol.
(*Batch_Gates_Generation)(void*,uint64_t ,void **) is the underlying protocol function, generates a batch of gates.
(*Gate_Check)(void*,uint64_t ,void **) is the underlying protocol function, checks a gate in a batch.
(*Gate_Copy)(void*,uint64_t ,void *,uint64_t ,void **) is the underlying protocol function, copies a gate in a batch to another gate in another batch.
(*Bucket_Build)(void*, uint64_t , void *, uint64_t *, void **) is the underlying protocol function, builds up a bucket given the indexes of the gates in it.
(*Evaluation)(void*, void *, void *, void *,uint64_t, void **) is the underlying protocol function, evaluates the bucket with given inputs.
(*MallocGatePointer)(void*, uint64_t ,void **) is the underlying protocol function, allocates the memory for a gate batch.
(*MallocBucketPointer)(void*, uint64_t ,void **) is the underlying protocol function, allocates the memory of a bucket.
(*Refresh_Seed)(void**,block) is the underlying protocol function, refreshes the random seed.
